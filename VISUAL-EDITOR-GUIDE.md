# Руководство по Visual Editor

## Доступ к редактору
Перейдите в админ-панель и нажмите кнопку "Visual Editor" или откройте напрямую: `visual-editor.html`

## Основные возможности

### 1. Переключение устройств
В верхней панели выберите размер экрана:
- **Desktop** (1400×800) - настольные компьютеры
- **Tablet** (768×1024) - планшеты
- **Mobile** (375×667) - смартфоны

### 2. Изменение порядка элементов

#### Метод 1: Drag & Drop (перетаскивание)
1. Выберите нужное устройство (Tablet или Mobile)
2. В левой панели найдите элемент, который хотите переместить
3. Зажмите левую кнопку мыши на значке ⋮⋮ (drag handle)
4. Перетащите элемент в нужную позицию
5. Отпустите кнопку мыши
6. Изменения автоматически сохранятся

#### Метод 2: Кнопки вверх/вниз
1. Наведите на элемент в списке
2. Нажмите ↑ для перемещения вверх или ↓ для перемещения вниз
3. Изменения автоматически сохранятся

### 3. Порядок элементов по умолчанию (Desktop)
На Desktop порядок элементов фиксированный и соответствует структуре HTML:
1. Логотип
2. Слоган
3. Услуги
4. Контакты
5. Карта

### 4. Индикатор порядка
Рядом с каждым элементом отображается номер позиции в текущем устройстве.

### 5. Live Preview
В центральной панели отображается предпросмотр сайта. Изменения порядка применяются в реальном времени.

### 6. Редактирование свойств элементов
1. Нажмите на элемент в списке или в превью
2. Справа откроется панель со свойствами
3. Измените нужные параметры (размер, цвет, шрифт и т.д.)
4. Нажмите "Сохранить изменения"

## Примеры использования

### Пример 1: На мобильном показать контакты первыми
1. Переключитесь на Mobile
2. Перетащите "Секция контактов" в самый верх списка
3. Теперь на телефонах контакты будут первыми

### Пример 2: На планшете карта после услуг
1. Переключитесь на Tablet
2. Перетащите "Карта" под "Секция услуг"
3. На планшетах карта теперь идет сразу после услуг

## Технические детали

### Как это работает
- Каждое устройство хранит свой порядок элементов в localStorage
- На главной странице JavaScript определяет размер экрана
- Применяется соответствующий порядок через CSS flexbox order
- При изменении размера окна порядок автоматически обновляется

### Ограничения
- Порядок можно менять только для основных секций (логотип, услуги, контакты, карта)
- На Desktop порядок фиксирован
- Некоторые элементы (header, footer) не могут быть перемещены

### Сброс настроек
Чтобы вернуть порядок по умолчанию:
1. Откройте консоль браузера (F12)
2. Выполните: `localStorage.removeItem('sewingData')`
3. Обновите страницу

## Горячие клавиши
- **Ctrl + S** - Сохранить изменения
- **Escape** - Закрыть панель свойств
- **Ctrl + Z** - Отменить (в разработке)
